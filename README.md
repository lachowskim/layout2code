# Strona Portfolio

Production portfolio website compiled into static assets in `dist/` using Nunjucks, modular SCSS, and vanilla JavaScript. Build pipeline is powered by Gulp.

## Features

- Template-based pages with Nunjucks (`src/templates/`, `src/pages/`)
- SCSS styling using partials and functions (`src/scss/`)
- Modular JavaScript (navigation, scroll effects, UI)
- Responsive layout across desktop and mobile
- Deployment-ready output in `dist/`

## Project Status

- Production-ready
- Active branch: `main`
- Deployment artifact: `dist/`

## Stack

- HTML5 + Nunjucks
- SCSS (Dart Sass)
- JavaScript (ES modules)
- Gulp 5 build pipeline
- PostCSS + Autoprefixer
- Terser + sourcemaps

## Key Implementation Points

- Template-based page composition from `src/templates/` and `src/pages/`
- Modular front-end scripts (`navigation`, `scroll`, `ui`, `effects`)
- Responsive layout and interaction patterns across desktop and mobile
- Build output generated into `dist/` for deployment

## Deployment

- Repository is published with full project contents (`src/` and `dist/`).
